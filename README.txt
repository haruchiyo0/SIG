SIGMA Flood - Web GIS Evakuasi Banjir

Cara menjalankan:
1. Extract ZIP ini.
2. Buka folder SIGMA_Flood_Website di VS Code.
3. Install extension Live Server.
4. Klik kanan index.html -> Open with Live Server.

Catatan:
- Data pada folder data/ masih simulasi untuk prototype.
- Data dapat diganti dengan hasil export GeoJSON dari QGIS.
- Struktur nama file sebaiknya tetap sama agar script.js tidak perlu diubah.
